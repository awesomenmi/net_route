Для того, чтобы изобразить __ассиметричный роутинг__, достаточно изменить файл /etc/quagga/ospfd.conf на одном из роутеров, добавив строчки:
```
interface vlanXX
ip ospf cost XXX
```


Пример содержимого файла для R1:
```
hostname R1

router ospf
    ospf router-id 192.168.12.1
    network 192.168.12.0/30 area 0
    network 192.168.13.0/30 area 0
    network 10.1.0.0/24 area 1

interface vlan13
ip ospf cost 300

log file /var/log/quagga/ospfd.log
```

Для __симметричного роутинга с дорогим линком__, необходимо изменить "стоимость" интерфейсов в одном VLAN-e, например в VLAN13, то есть для роутеров R1 и R3.

Пример содержимого файла для R3:
```
hostname R3

router ospf
    ospf router-id 192.168.13.2
    network 192.168.13.0/30 area 0
    network 192.168.23.0/30 area 0
    network 10.3.0.0/24 area 3

interface vlan13
ip ospf cost 300

log file /var/log/quagga/ospfd.log
```
